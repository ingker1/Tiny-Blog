# Tiny-Blog
基于四年写博客的经验，我做了这个系统，避开无用的功能，专注于真正提升写作体验。

自用的简单博客系统，使用四张核心表覆盖动态博客的基本功能。

既有动态博客的灵活、便利和实用性，又轻量化不臃肿。
